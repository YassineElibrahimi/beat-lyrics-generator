# SoundFonts for High-Quality Playback

To use FluidSynth for high-quality audio, download a SoundFont (.sf2) file and place it here.  
Recommended free SoundFonts for hip-hop:

- **SGM-V2.01** (excellent all-round) – [Download from Internet Archive](https://archive.org/details/sgm-v2.01-soundfont)
- **FluidR3 GM** – [Download from MuseScore](https://musescore.org/en/handbook/soundfonts-and-sfz-files#fluid-soundfont)

Rename the downloaded file to `default_soundfont.sf2` or modify the path in `cli.py`.
